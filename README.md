# Charity Website

This is a non-profit charity website designed to provide information about our mission, how to donate, and ways to get in touch with us.

## Project Structure

```
charity-website
├── src
│   ├── index.html          # Main HTML document
│   ├── css
│   │   └── styles.css      # Styles for the website
│   ├── js
│   │   └── main.js         # JavaScript for interactivity
│   ├── pages
│   │   ├── about.html      # About the charity
│   │   ├── donate.html     # Donation information and form
│   │   └── contact.html     # Contact information and form
│   └── assets
│       └── images          # Images used in the website
├── package.json            # npm configuration file
└── README.md               # Project documentation
```

## Setup Instructions

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/charity-website.git
   ```

2. Navigate to the project directory:
   ```
   cd charity-website
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Open the `src/index.html` file in your browser to view the website.
